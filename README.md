# Face Recognition

这是一个人脸识别项目。

## v1.0功能

- 人脸检测
- 人脸识别
- 对人脸进行命名
- 将命名后的人脸数据存入数据库
- 从数据库中快速调用人脸数据

## 使用

需要安装 Python 3 和以下库：

- opencv-python
- dlib
- sqlite3
- tkinter

运行 `src/face3.py` 文件即可开始使用。

```
 python3 ./src/face3.py
```
